The Ultimate Steak-Eating Tracker for Twitch Chat

SteakBot is a fun and interactive Twitch bot that brings a hilarious and bizarre steak-eating experience straight to your chat. Every time a viewer types !bettersteak, the bot serves them a completely random, often absurd steak variation—from a "crispy garlic steak" to a "steak grilled on a lightsaber".

With over 100+ unique steak types and a global steak counter for your chat, SteakBot turns eating virtual meat into a full-blown community event.

🔧 Features

  • Random Steaks: Type !bettersteak to get a wacky new steak each time.
  • Shared Meals: Type !bettersteak @username to share your steak with a friend.
  • Custom Steaks: Mods/streamers can use !addsteak steak name to add cursed new ideas permanently.
  • Personal Tracking: !mysteaks shows your personal count and last 5 steaks.
  • Community Tracker: SteakBot tracks how many steaks your entire chat has eaten together. Updated live!
  • Easy to Deploy: No database required. Uses local JSON and TXT files. Built with TwitchIO and .env config.
  • Lightweight HTTP Server: For easy uptime monitoring or pings.

⚠️ Warning: There could be bugs when using the !addsteak command that might cause custom steaks to disappear.
If you want to add steaks permanently, please send a DM with all the steaks you want to add to fablp1220.

📜 Commands

  • !bettersteak – Eat a random steak
  • !bettersteak @Bob – Share a steak with another user
  • !addsteak steak dipped in nacho cheese – Adds a new steak type (Mods only)
  • !mysteaks – Shows your steak count and recent steak history
